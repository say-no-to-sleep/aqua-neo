# Aqua Neo

Aqua Neo is a component showcase and self-contained Aqua-inspired UI framework.

The original style direction came from [thatdhruv/aqua2](https://github.com/thatdhruv/aqua2), but this version has been edited extensively.

## Skill Sync

This repo now tracks a copy of the Codex skill at `skills/aqua2-ui/SKILL.md`.

- Pull latest from Codex skill dir into repo:
  - `scripts/sync-skill.sh pull`
- Push repo version back to Codex skill dir:
  - `scripts/sync-skill.sh push`
